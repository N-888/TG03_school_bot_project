# 🎓 School Student Bot

Современный Telegram-бот на **Python + aiogram 3 + SQLite**, который по шагам запрашивает у пользователя **имя**, **возраст** и **класс**, а затем сохраняет данные в базу данных **school_data.db** в таблицу **students**.

---

## ✨ Что умеет бот

- красиво приветствует пользователя;
- показывает удобные, понятные кнопки и команды;
- работает пошагово через FSM;
- проверяет корректность введения возраста;
- сохраняет все данные в базу данных **school_data.db** в таблицу **students** в SQLite;
- позволяет отменить заполнение в любой момент;
- легко запускается локально в PyCharm на Windows.

---

## 🧱 Стек проекта

- Python 3.11+
- aiogram 3.x
- SQLite
- python-dotenv

---

## 🚀 Как запустить

1. Создай бота в Telegram
- Открой Telegram.
- Найди @BotFather.
- Отправь команду /newbot.
- Придумай имя и username для бота.
- Скопируй токен.

2. Подготовь файл .env

- Вставь туда через Терминал: BOT_TOKEN=сюда_вставь_токен_от_BotFather

3. Установи зависимости

- Вставь через Терминал: pip install -r requirements.txt

4. Запусти бота 

- Вставь через Телеграм: python app.py

---

## 🗃️ Структура базы данных

Таблица students создается автоматически и содержит поля:

- id — INTEGER, PRIMARY KEY, AUTOINCREMENT
- name — TEXT
- age — INTEGER
- grade — TEXT

## 💎 Сильные стороны проекта

- код логично разбит по папкам и файлам;
- настройки вынесены отдельно;
- база данных подключена по классической схеме;
- логика анкеты сделана через FSM, как в современных примерах aiogram 3;
- интерфейс дружелюбный: кнопки, эмодзи, подсказки, понятные сообщения;
- проект легко расширяется: 
    - можно добавить просмотр списка учеников, 
    - удаление записей, 
    - админ-панель, 
    - экспорт в Excel.
